# OneVoice 宣传方案

## 方案 A：隐私痛点（推荐主线）

**主标题**：你的声音，不该先上传再被听见。

**短文案**：

很多语音输入工具，第一步是把你的声音传到服务器。OneVoice 不一样：默认使用 Apple 设备端语音识别，录音和转写都可以离线完成。没有账号、没有广告、没有分析 SDK，也没有 OneVoice 的音频服务器。

免费、开源、离线。按住 Fn，说完即写入当前输入框。

**适合**：技术用户、隐私敏感用户、开源社区、Product Hunt / X。

## 方案 B：效率痛点（推荐大众传播）

**主标题**：打字跟不上想法，就别再打字。

**短文案**：

写邮件、回消息、记灵感、写提示词——按住 Fn 直接说，松开就出现在光标所在的位置。无需切换 App，不用复制粘贴，也不依赖网络。

OneVoice 是一款免费、开源、离线优先的全局语音输入工具。

**适合**：朋友圈、小红书、即刻、微信群。

## 方案 C：订阅疲劳（推荐传播钩子）

**主标题**：语音输入，不该又是一笔订阅。

**短文案**：

不按分钟收费，不限制每天次数，不把基础功能锁在会员后面。OneVoice 免费且开源：Mac 上全局语音输入，iPhone 上后台录音、自动转写、搜索和私有 iCloud 同步。

你的设备，你的模型，你的声音。

**适合**：对比型广告、社区发布、朋友推荐。
